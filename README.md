# Task Manager
A simple web-based task manager to create, track, and manage daily tasks efficiently.

<img width="1020" height="597" alt="image" src="https://github.com/user-attachments/assets/b0767dfd-609d-421e-b81d-fb495aa6fac2" />

## Features
- **Title & Description:** Add tasks with a title and description
- **Status Tracking:** Mark tasks as complete or pending
- **Due Date:** Set and view task deadlines
- **Actions:** Edit or delete tasks
  
- ## Technologies
- **Backend:** Laravel (PHP framework)  
- **Database:** PostgreSQL  
- **Frontend:** HTML, CSS, JavaScript  
- **IDE:** IntelliJ IDEA  
- **Deployment:** Railway

- ## Installation
1. Clone the repository:
 git clone  https://github.com/Kibriyoo/taskmanager
2. Navigate to the project folder:
cd task-manager
3. Configure your PostgreSQL database in the project files.  
4. Open `index.html` (or main file) in your browser to use the app.

   ## Usage
- Add a task by entering **Title**, **Description**, and **Due Date**.  
- Update **Status** as tasks progress.  
- Use **Actions** to edit or delete tasks.
